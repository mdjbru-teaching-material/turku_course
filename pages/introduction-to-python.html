Introduction to Python

Presentation of Python

  • Python is an interpreted language (distinction compiled/interpreted)

  • There are a lot of libraries available (e.g. Numpy, BioPython, matplotlib, pandas) and a large community
  • Python is multi-platform (if it works on your laptop, it should work on the CSC servers)
